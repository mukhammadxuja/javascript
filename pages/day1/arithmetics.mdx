## Arithmetics | Arifmetika

Keling, arifmetik amallarni Google Chrome konsolida _`console.log()`_ yordamida yozishni ko'proq mashq qilaylik. Matnga qo'shimcha ravishda biz JavaScript-dan foydalanib matematik hisob-kitoblarni ham amalga oshirishimiz mumkin. Keling, quyidagi oddiy hisob-kitoblarni qilaylik.
Google Chrome konsolida **_`console.log()`_** funksiyasisiz to'g'ridan-to'g'ri JavaScript kodini yozish mumkin. Biroq, text muharrirlarida bu yozish talab qilingani tufayli biz o'zimizni shunday yozishga o'rgatishimiz, mashq qilishimiz uchun **_`console.log()`_** funksiyasidan foydalanamiz.

![Arithmetic](images/arithmetic.png)

```js copy
console.log(2 + 3); // Qo'shish amali
console.log(3 - 2); // Ayirish amali
console.log(2 * 3); // Ko'paytirish amali
console.log(3 / 2); // Bo'lish amali
console.log(3 % 2); // Modulus | qoldiqni topish
console.log(3 ** 2); // Daraja 3 ** 2 == 3 * 3
```
