## Sharhlar

##### Comments | Sharhlar

Javascript kodimizga sharhlar qo'shishimiz mumkin. Kodni yanada tushunish oson bo'lish va kelajakda mavjud bo'ladigan xatoliklarni (bug) osonlik bilan bartaraf etish uchun zarur. JavaScript bizning kodimizning sharh (comment) qismini o'qimaydi. JavaScript-da `//` bilan boshlanadigan har qanday matn qatori sharhdir.

**Misol: Bir qatorli izoh**

```js copy
// Mening birinchi sharhim
// Mening ikkinchi sharhim
// Mening uchunchi sharhim
```

**Misol: Bir necha qatorli izoh**

```js copy
/*
  Bu ko'p qatorli sharh
  Ko'p qatorli sharhlar bir nechta satrlarni olishi mumkin
  JavaScript - bu internet tili
*/
```
