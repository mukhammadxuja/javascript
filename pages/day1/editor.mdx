## Code Editor

Kodlarimizni brauzer konsolida yozishimiz mumkin, ammo bu kattaroq loyihalar imkonsiz. Ushbu Javasript 30 kunda qo'llanmasida biz Visual Studio muhariridan foydalanamiz. Shu bilan birgalikda siz [Brackets](https://brackets.io/), [Sublime](https://www.sublimetext.com/), [Fleet](https://www.jetbrains.com/fleet/), [Webstrorm - Pullik](https://www.jetbrains.com/webstorm/) va shunga o'xshash matn muharirlaridan foydalanishingiz mumkin.

<Steps>
### Visual Studio Codeni yuklash

Visual Studio Code juda mashhur ochiq manbali matn muharriri bo'lgani uchun [Visual Studio Codeni yuklashni](https://code.visualstudio.com/) tafsiya qilaman, ammo siz yuqorida keltirilgan matn muharirlarini ishlatishingiz mumkin.

![Vscode](images/vscode.png)

Agar siz Visual Studio kodini o'rnatgan bo'lsangiz, undan foydalanishni boshlaymiz.

### Visual Studio Codedan foydalanish

Visual Studio belgisini ustiga ikki marta bosish orqali oching. Uni ochganingizda, siz bunday interfeysga ega bo'lasiz. Misol keltirigan rasmlar ketma ketligida, uni ishlatishga harakat qilib ko'ring.
