#### Google Chrome Consoleni ochish

You can open Google Chrome console either by clicking three dots at the top right corner of the browser, selecting _More tools -> Developer tools_ or using a keyboard shortcut. I prefer using shortcuts.

![Opening chrome](images/opening_developer_tool.png)

Chrome console ni ochish talab qilinadi va buni ochish uchun ushbu tugmalar ketma ketligni bosing.

```sh copy
Macbookda
Command+Option+J

Windows/Linux:
Ctl+Shift+J
```

![Consoleni ochish](images/opening_chrome_console_shortcut.png)

Google Chrome console ini ochganingizdan so'ng, belgilangan tugmalarni o'rganishga harakat qiling. Biz ko'p vaqtimizni console da o'tkazamiz. Console - bu sizning JavaScript kodingiz ketadigan joy. Google Console V8 dvigateli - JavaScript kodingizni mashina kodiga o'zgartiradi.
